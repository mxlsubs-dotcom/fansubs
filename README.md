# MXL Subs Website

Static website based on `Executive Summary.pdf`.

## Pages

- `index.html` - homepage and episode list
- `episode.html` - YouTube embed, subtitle toggle, VTT download
- `about.html` - project explanation, usage rules, contact/takedown

Open `index.html` in a browser or publish the folder with GitHub Pages.
